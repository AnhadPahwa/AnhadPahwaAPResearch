# Token-Based Privacy-Preserving Identity Prototype

This repository contains a research prototype developed for an AP Research project examining the feasibility of cryptographic token-based identity systems that balance child safety, fraud prevention and user anonymity.

## Structure
- issuer/: credential issuance and revocation
- wallet/: holder-side presentation logic
- verifier/: relying party verification logic
- experiments/: experimental evaluation harness
- examples/: protocol message formats

## Disclaimer
This prototype is for academic research only and does not interface with real users, networks or services.
